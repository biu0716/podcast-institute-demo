## 关键判断（防止走回头路）

- **全文转写是"增强项"不是"核心路径"**：1 期听稿 + 2 期简介也能产出有价值的跨集比对。
- **不逆向小宇宙登录 API**：搜索 API 需鉴权、是逆向接口，不适合开源；检索保留轻量 agent。
- **限流当常态**：默认只即时转标星 1 期；精读默认后台化、永不给倒计时。
- **不做**：多人队列 / 配额 / 共享缓存 / 付费逻辑 / 截图 OCR。
